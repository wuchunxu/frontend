三栏布局之圣杯布局

圣杯布局的要点

组成元素 容器，左，中，右三块
布局设计
  1. 容器设置左右padding，为两边的侧边栏腾出空间
  2. html中三块先后顺序为.main<<.left<<.right
  3. 中间部分宽度100%，左右宽度固定
  4. 左中右三块浮动和定位设置：{float:left;position:relative;}
  5. 设置margin-left值，将左和右拉回.main所在那一行
  6. 最后，设置left/right值，以调整左右位置

圣杯布局中涉及的主要知识点

  • margin负值的使用
  • left负值的使用

三栏布局之双飞翼布局

双飞翼布局的要点

组成元素 容器，左，中，右三块
布局设计
  1. 首先，HTML中按：中、左、右先后顺序写好标签
  2. 左中右三块均左浮动，而中宽度设置为100%，这样，左和右被挤到下一行
  3. 设置左侧margin-left值为-100%，右侧为负的自身宽度
  4. 此时，左右两侧会遮挡住中间的一部分，在中间一部分中再插入一个div，内部div用来显示内容。外部设置左右padding即可。

双飞翼布局与圣杯布局的区别

双飞翼布局不需要position来定位（左右调整）
